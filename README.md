# vps-vms
This is a VPS for different OS vms.


**Follow the steps to make your VPS**

# First Step :
Fork this repository

# Second Step :
create a Codespace after fork

# Third Step :
run this command :
```
sudo apt install qemu-system cloud-image-utils wget
```
# Last Step :
```
bash vms.sh
```

** Note: Now select as per you want **
